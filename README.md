# skriptimine-bash
Skriptimis aluste ülesanne "Skripti versioonihaldus Git ja Github näitel" esimene osa.

# Mida skript teeb?
- Käivitades skripti ütleb skript sulle "Tere" kasutades nime, mis on skripti kirjutatud (võib muuta kui on soovi).
- Lisaks ütleb ka tänase kuupäeva ning kellaaja ja kontrollib sudo käsuga uuendusi (küsib kasutaja käest parooli).
